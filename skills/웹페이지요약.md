# 절차

## 1. URL 식별
사용자 메시지에서 URL 추출. 없으면 사용자에게 URL을 물어보기.

## 2. 페이지 로드
1. `browser_open(URL)` 로 페이지 열기
2. `browser_get_text(max_chars=8000)` 로 가시 텍스트 추출
   - 너무 길면 max_chars를 늘리거나 두 번 호출

## 3. 핵심 추출
페이지 내용에서 다음을 분리:
- **제목/주제**: 첫 줄 또는 명시된 헤더
- **핵심 주장 3~5개**: 주요 단락에서 추출
- **인용할 만한 수치/날짜**: 있다면 그대로 보존
- **저자/출처**: 명시되어 있으면 표시

## 4. 보고서 작성
다음 형식으로 한글 응답:

```
📄 [페이지 제목]
출처: [URL]

🎯 핵심 요약 (3줄)
- ...
- ...
- ...

📌 주요 내용
1. ...
2. ...
3. ...

💡 시사점 (있다면)
- ...
```

## 5. 메모리 저장 옵션
사용자가 "기억해줘" 또는 "저장해줘" 라고 하면:
- `remember(f"요약-{날짜}-{제목}", 핵심 내용)` 로 저장

## 주의
- 페이지 내용을 그대로 베끼지 말고 반드시 자기 말로 요약
- 광고/네비게이션/푸터 텍스트는 제외
- 페이지가 비어있거나 로그인 필요면 사용자에게 알림
