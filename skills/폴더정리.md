# 절차

## 1. 대상 폴더 확인
1. 사용자가 폴더 경로를 명시했다면 그 경로 사용
2. 모호하면 기본값 `C:\Users\lkh\Downloads` 사용
3. `list_directory(경로)` 로 내용 확인

## 2. 분류 계획 수립
파일을 확장자로 그룹화:
- 이미지: .png, .jpg, .jpeg, .gif, .bmp, .webp
- 문서: .pdf, .docx, .doc, .txt, .md, .hwp
- 스프레드시트: .xlsx, .xls, .csv
- 압축: .zip, .7z, .rar, .tar, .gz
- 코드: .py, .js, .ts, .html, .css, .json
- 영상: .mp4, .mkv, .avi, .mov
- 음악: .mp3, .wav, .flac

## 3. 사용자 확인
실행 전 반드시 사용자에게 어떤 분류로 어디로 이동할지 보고하고 동의 받기.

## 4. 실행
`run_powershell` 로 PowerShell 명령 실행. 예:
```
Get-ChildItem 'C:\Users\lkh\Downloads' -File |
  Where-Object { $_.Extension -in '.png','.jpg','.jpeg' } |
  Move-Item -Destination 'C:\Users\lkh\Downloads\images' -Force
```

폴더가 없으면 먼저 생성:
```
New-Item -ItemType Directory -Path 'C:\Users\lkh\Downloads\images' -Force
```

## 5. 결과 보고
- 이동한 파일 수
- 분류별 통계
- 정리 후 폴더 구조 (`list_directory` 로 재확인)

## 주의
- **파일 삭제는 절대 자동으로 하지 말 것**. 이동(Move-Item)만 허용.
- 시스템 폴더(C:\Windows, C:\Program Files 등)는 거부.
