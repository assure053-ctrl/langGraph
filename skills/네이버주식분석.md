# 절차

## 1. 종목 검색
1. `browser_open("https://finance.naver.com")` 으로 네이버 금융 메인 페이지 열기
2. `browser_get_inputs()` 로 검색창 셀렉터 확인 (보통 `input[name="query"]`)
3. `browser_type(셀렉터, 종목명, press_enter=True)` 로 검색 실행
4. `browser_url()` 로 종목코드 추출 (URL 형식: `?code=000660`)

대안: 검색 실패 시 `browser_open("https://finance.naver.com/search/searchList.naver?query=종목명")` 으로 직접 이동

## 2. 외국인·기관 매매동향 수집
1. `browser_open(f"https://finance.naver.com/item/frgn.naver?code={코드}")` 로 이동
2. `browser_get_text()` 로 페이지 내용 가져오기
3. 표에서 최근 5일치 데이터 추출:
   - 날짜, 종가, 등락률, 외국인 순매수, 기관 순매수

## 3. 최신 뉴스 수집
1. `browser_open(f"https://finance.naver.com/item/news.naver?code={코드}")` 로 이동
2. `browser_evaluate` 로 뉴스 iframe(`news_news.naver`) 안의 `td.title a` 텍스트 추출
   - 가능하면 `browser_get_text()` 로도 가져와 보고 안 되면 evaluate 사용
3. 최신 뉴스 제목 5개 수집

## 4. 보고서 작성
다음 두 섹션으로 한글 보고서 작성:

**1. 외국인·기관 매매 동향 요약** (3~4문장)
- 5일간 추세, 최근 1일 흐름, 가격과의 상관관계

**2. 뉴스 기반 시장 분위기** (3~4문장)
- 긍정/부정/중립 판단 + 근거 뉴스 인용

## 5. 사용자 컨텍스트 활용
- 시작 전 `recall("종목")` 또는 `recall("관심")` 으로 사용자의 관심 종목 확인
- 사용자가 특정 종목을 자주 분석한다면 `remember("관심종목", "...")` 으로 저장
